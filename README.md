1. Opis projektu:

Aplikacja webowa do zarządzania i analizy danych NBA. Umożliwia przeglądanie zespołów, zawodników, meczów, kontraktów i kontuzji. System automatycznie synchronizuje dane z zewnętrznym API NBA, generuje zaawansowane raporty analityczne oraz wizualizuje statystyki.

2. Technologie
Backend:

- Django
- PostgreSQL (relacyjna baza danych)
- NBA API (oficjalne dane NBA)
- psycopg2 (adapter PostgreSQL)

Frontend:

- HTML5 + CSS3
- JavaScript
- Bootstrap 5.3 (framework CSS)
- Font Awesome 6 (ikony)

Architektura:

- Multi-Page Application
- Wzorzec MTV (Model-Template-View)
- RESTful endpoints
